# Unreal 게임개발종합반 1-1 ~ 1-3

---

## Unity 와 Unreal 차이점

- Unity → C# 사용  
- Unreal → C++ 사용  

- Unity  
  - 모바일 / 멀티 플랫폼 강함  
  - 인디게임, 모바일 게임에 많이 사용  

- Unreal  
  - AAA급, 고사양 게임에 많이 사용  
  - 예: 포트나이트, 기어즈 오브 워  

---

## 서버란?

- 여러 플레이어가 동시에 접속해도 공정하고 일관성 있게 게임 진행을 담당하는 핵심 시스템

### 웹 서버
- 정적인 콘텐츠 전송 (웹페이지, 이미지 등)
- HTTP / HTTPS 요청 처리

### 게임 서버
- 실시간 상호작용 처리
- 많은 사용자 상태와 게임 데이터를 동시에 관리
- 성능 중요

---

## 프로토콜이란?

- 컴퓨터 간 정보를 교환할 때 따르는 규칙 (통신 규약)

### 주요 프로토콜

- HTTP / HTTPS → 웹 데이터 전송  
- TCP / UDP  
  - UDP → 실시간 게임에 유리  
  - TCP → 안정성 중요할 때 사용  

- 기타  
  - WebSocket → 브라우저 실시간 통신  
  - RUDP → 신뢰성 강화 UDP  

---

## 클라이언트

- 화면 출력 담당  
- 사용자 입력(키보드/마우스/터치) 수신  
- 서버 결과를 그래픽/사운드로 표시  

---

## 네트워크 구조

### P2P
- 중앙 서버 없이 플레이어끼리 직접 연결  
- 서버 비용 낮음  
- ✂ 해킹 방어 / 속도 문제 발생 가능  

### 하이브리드
- 서버 + P2P 혼합 구조  
- 상황에 따라 사용  

### 단일 서버
- 하나의 서버가 모든 기능 담당  
- ✂ 사용자 많아지면 과부하  

### 분산 서버
- 여러 서버가 역할 분담  
- 로그인 / 채팅 / 전투 등 분리  
- 대규모 게임에 적합

# 퀘스트

---

## 1️⃣ Unreal Engine 조사

### 게임 엔진이란?

- 그래픽 / 물리 / 사운드 / 입력 등을 통합 제공하는 개발 도구

### Unreal Engine이란?

- 에픽게임즈가 개발한 실시간 3D 제작 및 게임 엔진  
- 게임 외 영화 / 건축 / 시뮬레이션 등 다양한 분야 활용  

---

## Unreal Engine 주요 기능 (발로란트 예시)

- 그래픽 렌더링  
  - 스킬 이펙트 대비 강조  

- 물리 엔진  
  - 총알 판정 / 스킬 충돌 처리  

- 비주얼 스크립팅  
  - 문 열림 / 캐릭터 스킬 로직  

---

## Unreal 제작 대표 게임

### 포트나이트
- 차세대 그래픽 기술 적용  
- 100명 동시 접속 멀티플레이 구현  

### 배틀그라운드
- 오픈월드 스트리밍 / LOD 시스템 활용  
- 차량 / 총기 / 충돌 물리 처리  

### 파이널 판타지 7 리메이크
- 실시간 렌더링 컷신  
- 고품질 캐릭터 표현  
- 애니메이션 시스템 활용  

---

## 2️⃣ 게임 레벨(Level)

### 레벨 역할

- 게임 진행 단위  
- 시간 / 공간 구분  
- 난이도 조절  
- 스토리 전달  

---

## Unreal 레벨 구성 요소

### World
- 레벨 전체 관리  
- 모든 Actor 존재 공간  

### Actor
- 레벨 내 모든 객체 기본 단위  
- 캐릭터 / 아이템 / 건물 등  

### Light
- 밝기 / 분위기 결정  
- Directional / Point / Spot Light  

### Camera
- 플레이어 시점 담당  
- 1인칭 / 3인칭 / 고정 카메라
